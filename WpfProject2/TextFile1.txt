Элементы управления WPF
Текстовые элементы управления(могут содержать только текс):
1.TextBlock
Предназначен для вывода текстовой информации
<TextBlock>Какой то текст</TextBlock>
<TextBlock = "Другой текст"/>
Может содержать вложенные элементы Run,для разбиения текста на составляющие
Свойства TextBlock
а) Свойство LineHeight- позволяет указывать высоту строк
б)Свойство TextWrapping - позволяет переносить текст. Может принимать значения:
 Wrap - текст всегда разрывается на краю элемента управления,даже если придется переносить слишком
 длинные слова
 WrapWithOverflow - позволяет некоторым строкам выйти за правый край,если не найдеться 
 место для рвазбиения строки
в)Свойство TextAligmrentвыравнивает по центру(Center), правому(Right),левому(Left)
Элемент lineBreak позволяет перенести текст на новую строку

2.TextBox
Предназначен для вывода или ввода текстовой информации
Поддерживает все свойства  доступные TextBlock
Свойства TextBox
а)MaxLength - позволяет задать максимальное колличество вводимых символов
б)MinLines и MaxLines - определяют минимальное и максимальное колличество строк,которые отображаются
на игровом поле.Они не влияют на колличество символов,только на колличество отображаемых строк
в)AcceptReturn - позволяет активировать переход на следущую строку при нажатии Enter
г)AcceptTab - позволяет активировать возможность создания табуляции при помощи Tab
д)VerticalScrollBarVisibility и HorizontScrollBarVisibility -отображают вертикальную и горизонтальную полосы прокрутки
е)IsReadOnly - позволяет заблокировать TextBox для редактирования,оставаив тоько для чтения

4.PasswordBox
Предназначен для ввода парольной информации,введенный текст скрывается при помощи маски(символов)
PasswordBox не поддерживает работу с буфером обмена
Свойства PasswordBox:
а)PasswordChar - учтанавливает символ маски,отображаемый при вводе пароля

Элементы управления списками
1.ListBox
Предназначен для создания списков переменной дляины, в которых пользователь может выбирать элементы
Свойство SelectionMode позволяет активировать множественный выбор
-Режим Multiple- можнот выбрать любой элемент или отменить его выбор,щелкнув по нему
-режим Extended - необходимо зажать клавишу Shift или Ctrl, для выбора нескольких элементов
-режим Single - допускает выделение только одного элемента 
2.ComboBox
Предназначен для отображения коллекци элементов в виде выпадающего списка

В качестве элементов ComboBox так же как и ListBox поддерживает практически любые элементы WPF

Наиболее эффективным является применение элемента ComboBoxItem

Свойство IsSelected позволяет выделить данный элемент как выбранный при запуске приложения
Свойство IsEnabled позволяет установить данный элемент как недоступый для выбора

У всех списков есть два способа заполнения элементов:
-добавление элементов прямо в коллекцию items с помощью кода или XAML
-привязка данных - более сложный способ, часто используется при работе с БД

Элементы выбора из диапазона значений
Данные элементы принимают числовое значение, которое находиться между заданными минимальными и максимальными значениями
Все элементы имеют общие свойства:
Value - текущее значение элемениа
Maximum и  Minimum - -верхний и нижний предел допустимвх значений
SmallCange и LargeCange - величина на которую увеличивается или уменьшается значение Value
Элемент Slider - ползунок позволяет устанавливать числовое значение используя ползунок
-Свойство Orientation - устанавливает ориентацию элемента
-Свойство IsSnapToTipPlacment
-TickPlacement
-Ticks

Элемент ProgressBar - индикатор прогресса.

Элемент Calendar - календарь
Свойство DisplayMode - задает формат отображения датю
Может быть:
Month - отображает дни месяца
Decade - отоброжает все года текущего десятилетия
Year - отброжает все месяцы текущего года
Свойство SelectionMode - определяет способ выделения дат.
Может быть:
- SingleDate(можно выбрать одну дату)
- None(нельзя выбрать дату)
- SingleRange(по нажатию Ctrl выделяет несколько последовательных дат)
- MultipleRange(по нажатию Ctrl выделяет несколько непоследовательно идущих диапазонов  дат)

Элемент DatePicker - ввод даты
Предостовляет текстовое поле для ввода даты с выпадающим календарем
Свойство IsDropDownOpen - позволяет оставить календарь открытым после выбора даты
Свойство SelectDateFormat - определяет формат даты
Short
Long

Элемент ScrollViewer - обеспечивает прокрутку содержимого
Свойства HorizontalScrollBarVisability и VerticallScrollBarVisability - задают активность горизонтальной и вертикальной прокрутки
Свойство CanContentScroll - активирует прокрутку по элементам

Элемент ViewBox
может содержатьодин дочерний элемент и масштабирует его в зависимости от размеров окна
Свойство Strech - позволяет указать как в занимаемой области должен масштабироваться элемент
StrechDirection - позволяет указать какие операции разрешены

Элемент Image 
Предназначен для работы с изображениями
Свойство Source - позволяет задать путь к изображению
Для начала работы с изображениями их нужно разместить внутри проекта  и каждому изображению в свойствах 
настроить тип действия при сборке ресурс

Элемент TabControl
Позволяет разделить интерфейс на несколько разных зон, которые можно выбрать, нажав на заголовок,который обычно находиться
сверху управления

Элемент ToolBar
Представляет собой панель инструментов.

Элемент WPFToolBar 
Обычно помещается внутри контейнера ToolBarTray
Позволяет перемещать элементы пользователю
